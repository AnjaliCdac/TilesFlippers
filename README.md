# 🎮 Tiles Flippers

An interactive puzzle game developed using **React Native** where players strategically occupy tiles on a game board to maximize connected edges and score points. The game supports both **Single Player (vs Computer)** and **Two Player** modes.

> Developed as part of **Game of Codes 2023**

# ✨ Features

- 🎯 Single Player (vs Computer)
- 👥 Two Player Mode
- 🔲 3×3 Grid (Level 1)
- 🔳 4×4 Grid (Level 2)
- 🧮 Automatic Score Calculation
- 📈 Bonus Points for Vertical, Horizontal & Diagonal Connections
- 🏆 Winner Detection
- 📖 Instructions Screen
- 🎨 Attractive UI
- 📱 Android Support

---

# 🕹️ Game Rules

Players take turns selecting tiles on the game board.

Points are awarded based on:

- Common edges
- Vertical lines
- Horizontal lines
- Diagonal lines

The player with the highest score at the end wins the game.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AnjaliCdac/TilesFlippers.git
```

Go to project

```bash
cd TileFlipers_ReactNative
```

Install dependencies

```bash
npm install
```

Run application

```bash
npx react-native run-android
```
